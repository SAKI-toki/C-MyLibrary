★SAKIライブラリ

☆SAKIライブラリとは？
	　プログラミング言語であるC++の自作汎用ヘッダオンリーライブラリです。
	名前の由来は『咲〜saki〜』というアニメが好きだからです。


☆使用方法
	　C++11以降の環境での使用を想定しているため、それ以前のC++03や98には対応していません。
	　C++14,17,20でのみ使用可能な関数の場合は、コメント等で記入しますが、
	C++11で使用可能なものは特に明記しませんのでご了承ください。

	　ヘッダオンリーなので、必要なものをインクルードし、
	名前空間が「saki」なので、使用するするときはsaki::〇〇と入力してください。
	例：　saki::method();
	　　　class hoge:public saki::foo<hoge>{};

☆環境
	 Visual Studio2017のC++で作成、動作確認しています。
	プロパティで[C/C++]->[言語]->[C++言語標準]を最終草案
				   ->[準拠モード]をいいえ　　を設定してください。


☆リファレンス
